#### 🏷️ 二级分类筛选
- 主分类选择后显示子分类标签
- 子分类标签显示资料数量
- 紫色高亮选中状态
- 响应式横向滚动

- 当选择主分类且子分类为"全部"时，按子分类分组显示
- 每个分组显示标题和数量

## 🚀 使用方法

### 1. 浏览所有资料
- 默认显示"全部"分类
- 所有资料按添加顺序显示

### 2. 按主分类筛选
- 点击主分类标签（如"浙江中文导游词"）
- 自动显示该分类下的所有资料
- 按子分类分组显示

### 3. 按子分类筛选
- 选择主分类后，点击子分类标签
- 只显示该子分类下的资料
- 不再分组显示

### 4. 搜索资料
- 输入关键词搜索
- 支持标题、描述、主分类、子分类
- 实时更新结果
